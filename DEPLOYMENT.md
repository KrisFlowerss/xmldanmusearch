# 弹幕搜索工具 - Vercel 部署指南

## Vercel 部署限制及解决方案

### 🚨 主要限制

1. **执行时间限制**
   - 免费版：10秒
   - Pro版：60秒
   - 解决方案：设置30秒超时限制

2. **内存限制**
   - 免费版：1024MB
   - Pro版：3008MB
   - 解决方案：限制文件大小和弹幕数量

3. **文件大小限制**
   - 请求体：最大50MB
   - 响应体：最大50MB
   - 解决方案：前端预检查文件大小

4. **无状态环境**
   - 全局变量在请求间不保持
   - 解决方案：每次请求重新处理数据

### 📋 部署前检查清单

1. **文件准备**
   - [ ] 确保XML文件小于50MB
   - [ ] 弹幕数量不超过10万条
   - [ ] 文件格式正确（标准B站XML格式）

2. **性能考虑**
   - [ ] 大文件可能需要更长处理时间
   - [ ] 复杂搜索可能超时
   - [ ] 建议分割大文件处理

3. **网络环境**
   - [ ] 确保网络连接稳定
   - [ ] 上传速度足够（50MB文件需要时间）

### 🎯 推荐使用方式

1. **文件大小建议**
   - 小于10MB：最佳性能
   - 10-30MB：良好性能
   - 30-50MB：可能较慢但可用
   - 大于50MB：需要分割文件

2. **搜索策略**
   - 使用具体关键词（避免过于宽泛）
   - 分批处理大量数据
   - 优先搜索热门时间段

### 🛠️ 故障排除

#### 常见错误及解决方案

1. **"搜索超时"**
   - 文件过大，尝试使用更小的文件
   - 网络不稳定，检查连接

2. **"文件处理失败"**
   - XML格式错误，检查文件格式
   - 文件损坏，重新下载XML

3. **"数据量过大"**
   - 弹幕数量超限，使用较短时间段的文件
   - 分割大文件处理

4. **"网络连接失败"**
   - 检查网络连接
   - 稍后重试
